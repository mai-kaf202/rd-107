# Расчёт энергетической увязки

## Общие входные данные
Согласно задания:
- $\alpha$ -- коэфф. избытка окислителя (соотношение компонентов);
- $P$ -- тяга;
- $p_к$ -- давление в камере сгорания;
- $F$ -- степень расширения сопла.
